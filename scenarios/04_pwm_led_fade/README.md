# Σενάριο 4: Αυξομείωση Φωτεινότητας LED (PWM Fade)

Σε αυτό το σενάριο, θα μάθουμε πώς να ελέγχουμε τη φωτεινότητα ενός LED χρησιμοποιώντας τη μέθοδο PWM (Pulse Width Modulation - Διαμόρφωση Εύρους Παλμού). 

## Στόχοι
- Κατανόηση της έννοιας του PWM.
- Χρήση της βιβλιοθήκης RPi.GPIO για τη δημιουργία σημάτων PWM.
- Δημιουργία ενός εφέ "αναπνοής" (fading) σε ένα LED.

## Το Κύκλωμα
Χρησιμοποιούμε το **GPIO 18** (Φυσικό Pin 12), το οποίο υποστηρίζει hardware PWM στο Raspberry Pi:
1. **GPIO 18 (Φυσικό Pin 12)** συνδέεται σε αντίσταση **330Ω** (πορτοκαλί καλώδιο).
2. Η άλλη άκρη της αντίστασης συνδέεται στην άνοδο ενός **Μπλε LED**.
3. Η κάθοδος του LED συνδέεται στη γείωση (GND - Pin 6).

## Οδηγίες
1. Διαβάστε τον κώδικα και παρατηρήστε πώς δημιουργείται το αντικείμενο `GPIO.PWM`.
2. Η εντολή `ChangeDutyCycle` δέχεται τιμές από 0 έως 100, ελέγχοντας πόσο "ποσοστό" του χρόνου το pin είναι ενεργό, άρα και τη φωτεινότητα.
3. Πατήστε **Run Code** και παρατηρήστε το LED.
4. Πειραματιστείτε αλλάζοντας τη συχνότητα του PWM από 100Hz σε 10Hz στην εντολή `GPIO.PWM(led_pin, 10)` για να δείτε το LED να τρεμοπαίζει!
