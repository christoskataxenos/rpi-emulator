# Σενάριο 7: Αναβόσβημα LED με C (Blink LED in C)

Καλώς ήρθατε στο πρώτο σας πείραμα προγραμματισμού μικροελεγκτή σε γλώσσα C! Σε αυτό το σενάριο, θα μάθουμε πώς να χρησιμοποιούμε C/C++ (στυλ Arduino ή STM32 HAL) για να αναβοσβήνουμε μια δίοδο LED που συνδέεται στο GPIO 17 (φυσικό Pin 11).

## Στόχοι
- Κατανόηση της δομής ενός προγράμματος C με τις συναρτήσεις `setup()` και `loop()`.
- Χρήση της `pinMode()` για τη ρύθμιση της εξόδου.
- Χρήση της `digitalWrite()` και της `delay()` για τον έλεγχο της κατάστασης και του χρόνου.
- Κατανόηση του πώς ο εξομοιωτής μεταφράζει live τον κώδικα C σε Python (C-to-Python Transpiler).

## Το Κύκλωμα
Το κύκλωμα είναι ήδη συνδεδεμένο:
1. Το pin **GPIO 17** συνδέεται στην αντίσταση 330Ω.
2. Η αντίσταση συνδέεται στην άνοδο του LED.
3. Η κάθοδος του LED συνδέεται στη γείωση (GND).

## Οδηγίες
1. Πατήστε το κουμπί **Run Code** για να μεταγλωττιστεί και να εκτελεστεί ο C κώδικας σας.
2. Δοκιμάστε να αλλάξετε το χρόνο καθυστέρησης στη `delay(1000)` σε `delay(200)` για να δείτε το LED να αναβοσβήνει πιο γρήγορα!
3. Παρατηρήστε πώς ο Monaco Editor άλλαξε αυτόματα τη σύνταξη και το highlighting σε C/C++ (cpp)!
