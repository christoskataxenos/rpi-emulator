# Σενάριο 3: Φανάρι Κυκλοφορίας (Traffic Light)

Σε αυτό το σενάριο, θα χρησιμοποιήσουμε 3 LEDs (Κόκκινο, Κίτρινο, Πράσινο) για να προσομοιώσουμε τη λειτουργία ενός φαναριού κυκλοφορίας.

## Στόχοι
- Διαχείριση πολλαπλών εξόδων GPIO ταυτόχρονα.
- Χρήση ακολουθιών με διαφορετικούς χρόνους αναμονής (`time.sleep()`).
- Κατανόηση βασικής λογικής αυτοματισμού.

## Το Κύκλωμα
Χρησιμοποιούμε τρία διαφορετικά GPIO pins, το καθένα συνδεδεμένο σε ένα LED μέσω μιας αντίστασης 330Ω:
1. **Κόκκινο LED:** Συνδέεται στο **GPIO 17** (Φυσικό Pin 11).
2. **Κίτρινο LED:** Συνδέεται στο **GPIO 27** (Φυσικό Pin 13).
3. **Πράσινο LED:** Συνδέεται στο **GPIO 22** (Φυσικό Pin 15).
Όλα τα LEDs γειώνονται (GND) σε διαφορετικά pins (Pin 6, 14, 20).

## Οδηγίες
1. Μελετήστε τον κώδικα και παρατηρήστε πώς ανάβουν με τη σειρά τα LEDs.
2. Πατήστε το **Run Code** για να δείτε το φανάρι σε δράση.
3. Δοκιμάστε να προσθέσετε τη λογική όπου πριν ανάψει το πράσινο, ανάβει το κόκκινο και το κίτρινο μαζί για 1 δευτερόλεπτο, όπως συμβαίνει στα πραγματικά φανάρια πολλών χωρών!
