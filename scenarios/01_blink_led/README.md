# Σενάριο 1: Αναβόσβημα LED (Blink LED)

Καλώς ήρθατε στο πρώτο σας πείραμα με το Raspberry Pi! Σε αυτό το σενάριο, θα μάθουμε πώς να συνδέσουμε μια δίοδο LED και να την προγραμματίσουμε να αναβοσβήνει χρησιμοποιώντας τη βιβλιοθήκη **RPi.GPIO**.

## Στόχοι
- Κατανόηση της διαφοράς μεταξύ της φυσικής αρίθμησης (BOARD) και της αρίθμησης GPIO (BCM).
- Ρύθμιση ενός GPIO pin ως εξόδου (OUTPUT).
- Χρήση της `time.sleep()` για τον έλεγχο του χρόνου.

## Το Κύκλωμα
Το κύκλωμα είναι ήδη προ-συνδεδεμένο στην πλακέτα:
1. **GPIO 17 (Φυσικό Pin 11)** συνδέεται στην είσοδο μιας αντίστασης **330Ω** (πορτοκαλί καλώδιο).
2. Η άλλη άκρη της αντίστασης συνδέεται στην **άνοδο (μακρύ πόδι)** του LED (κόκκινο καλώδιο).
3. Η **κάθοδος (κοντό πόδι)** του LED συνδέεται στη **γείωση (GND - Φυσικό Pin 6)** (μαύρο καλώδιο).

> [!NOTE]
> Η αντίσταση 330Ω είναι απαραίτητη για τον περιορισμό του ρεύματος. Χωρίς αυτήν, το LED θα διαρρεόταν από υπερβολικό ρεύμα και θα μπορούσε να καεί (θα δείτε σχετική προειδοποίηση στον εξομοιωτή αν την αφαιρέσετε!).

## Οδηγίες
1. Πατήστε το κουμπί **Run Code** για να δείτε το LED να αναβοσβήνει.
2. Δοκιμάστε να αλλάξετε το χρόνο καθυστέρησης στην `time.sleep(1.0)` σε `time.sleep(0.2)` για να δείτε το LED να αναβοσβήνει πιο γρήγορα!
